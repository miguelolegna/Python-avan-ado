Tecnologias Utilizadas

Java

Para o desenvolvimento da interface de backend e da lógica do assistente.

C++

Para processamento de sinal, como manipulação de áudio, se necessário, ou para integração de hardware.

Python

Para o processamento de dados e interações com APIs externas, especialmente para aprendizado de máquina se precisar de análise de dados.

Banco de Dados (DB)

Aqui você pode adicionar informações sobre o sistema de gerenciamento de banco de dados que será utilizado.

React

O React será responsável pela criação da interface de usuário (UI) do projeto, permitindo que os usuários interajam com o assistente virtual de forma intuitiva.
