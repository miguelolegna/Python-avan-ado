Java

O que é?

Java é uma linguagem de programação orientada a objetos amplamente utilizada no desenvolvimento de aplicativos para dispositivos móveis, aplicações web, e sistemas corporativos. Com seu slogan "Write Once, Run Anywhere" (Escreva uma vez, execute em qualquer lugar), Java permite que o código seja executado em diferentes plataformas, desde que a JVM (Java Virtual Machine) esteja instalada.

Como funciona?

Ao programar em Java, você escreve o código em um arquivo de texto com a extensão .java. Esse código é compilado em bytecode, que pode ser executado pela JVM. Essa abordagem garante que o mesmo código Java funcione em qualquer sistema que tenha a JVM instalada, independentemente do sistema operacional.

Características principais:

  • Orientação a Objetos: Facilita a organização do código e a reutilização através de classes e objetos.
  • Segurança: Java é projetado para ser seguro, com recursos que protegem contra malware.
  • Multithreading: Permite que múltiplas partes de um programa sejam executadas simultaneamente, aumentando a eficiência.

Exemplo prático:

Atividade: Criar um aplicativo simples.

Vamos criar um aplicativo que calcula a soma de dois números fornecidos pelo usuário.

Código Exemplo:


    import java.util.Scanner;

    public class Soma {
        public static void main(String[] args) {
            Scanner scanner = new Scanner(System.in);

            System.out.print("Digite o primeiro número: ");
            int num1 = scanner.nextInt();

            System.out.print("Digite o segundo número: ");
            int num2 = scanner.nextInt();

            int soma = num1 + num2;
            System.out.println("A soma é: " + soma);

            scanner.close();
        }
    }

Resumo e Conclusão:

Java é uma linguagem poderosa que se adapta a diversas necessidades de desenvolvimento. A prática de programação em Java ajuda a entender conceitos fundamentais de lógica de programação, estruturas de controle, e orientação a objetos.
