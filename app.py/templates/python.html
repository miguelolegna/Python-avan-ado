Python

O que é?

Python é uma linguagem de programação de alto nível, amplamente utilizada por sua simplicidade e legibilidade.

Como funciona?

Em Python, você escreve código em arquivos com a extensão .py. O código é interpretado por um interpretador Python, que executa as instruções uma por uma.

Características principais:

  • Fácil de aprender: A sintaxe simples e clara permite que iniciantes aprendam rapidamente.
  • Versatilidade: Usado em diversas áreas, como web, ciência de dados, e automação.

Exemplo prático:

Vamos desenvolver um jogo de adivinhação, onde o computador escolhe um número e o jogador tenta adivinhar.


import random

def adivinhacao():
    numero = random.randint(1, 10)
    tentativas = 0

    while True:
        guess = int(input("Adivinhe o número de 1 a 10: "))
        tentativas += 1

        if guess == numero:
            print("Você acertou! O número era:", numero)
            print("Você precisou de", tentativas, "tentativas.")
            break
        else:
            print("Tente novamente!")

adivinhacao()

Resumo e Conclusão:

Python é uma linguagem poderosa e acessível, ideal para quem está começando na programação.
