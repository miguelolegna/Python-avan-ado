História do Java

Java é uma linguagem de programação de alto nível, orientada a objetos, criada por James Gosling e sua equipe na Sun Microsystems em 1995. A filosofia do Java é "escreva uma vez, execute em qualquer lugar", permitindo que programas Java sejam executados em diversas plataformas.

Características do Java

Java é conhecido por sua portabilidade através da JVM (Java Virtual Machine), segurança, robustez, e suporte a programação concorrente.

Exemplo de Código em Java

O seguinte código é um exemplo de um programa Java que imprime "Hello, World!": 


public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Evolução do Java

Java passou por várias atualizações, cada uma introduzindo novas funcionalidades, como expressões lambda no Java 8.
