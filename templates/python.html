Exemplo prático:

    
Atividade: Criar um jogo simples.

    
Vamos desenvolver um jogo de adivinhação, onde o computador escolhe um número e o jogador tenta adivinhar.


    
Código Exemplo:

    


import random

def adivinhacao():
    numero = random.randint(1, 10)
    tentativas = 0

    while True:
        guess = int(input("Adivinhe o número de 1 a 10: "))
        tentativas += 1

        if guess == numero:
            print("Você acertou! O número era:", numero)
            print("Você precisou de", tentativas, "tentativas.")
            break
        else:
            print("Tente novamente!")

adivinhacao()
